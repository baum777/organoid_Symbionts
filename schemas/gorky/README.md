# Gorky JSON Schemas

JSON Schema definitions for pipeline data. Use for validation, documentation, and API contracts.

| Schema | Description |
|--------|-------------|
| `mention_signal.schema.json` | Raw X API mention |
| `normalized_event.schema.json` | CanonicalEvent (normalized) |
| `safety_result.schema.json` | Pre-generation safety filter output |
| `relevance_result.schema.json` | Relevance score + components |
| `narrative_result.schema.json` | Narrative label + confidence |
| `pattern_selection.schema.json` | Pattern selection result |
| `format_decision.schema.json` | Format decision (skip/short/expanded/thread) |
| `llm_generation_request.schema.json` | LLM request structure |
| `validation_result.schema.json` | Post-generation validation |
| `publish_result.schema.json` | Publish outcome |
| `analytics_log.schema.json` | Analytics event for dashboards |
