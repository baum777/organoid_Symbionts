# Autopost Strategy (GORKY)

## Goal
High-frequency, low-effort, high-shareability commentary that:
- roasts CT theater (rugs, wash volume, fake narratives)
- keeps replies short
- avoids targeted harassment / hate content
- occasionally drops lore "evil revival" lines

## Formats
1) One-liners (best)
2) 2-tweet mini-thread (rare)
3) "Chart autopsy" headline
4) "Court verdict" headline

## Cadence
- 1–3/day max (avoid spam)
- Use "series" files in `/autopost/` for rotating themes.

## Safety rails
- No doxxing
- No slurs
- No calls for harassment
- Mock behaviors/narratives, not protected classes or private individuals
