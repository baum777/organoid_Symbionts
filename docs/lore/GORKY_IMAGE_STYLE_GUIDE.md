# GORKY Image Style Guide

This document defines the mandatory visual style for all generated Gorky images.

The goal is strict visual consistency with the Gorky profile picture.

The profile picture style is:

- neon doodle
- glowing chalk lines
- simple shapes
- black background
- playful chaotic creature
- minimal detail
- meme readability

All generated images must follow this style.

---

## Core Visual Identity

Gorky is a chaotic neon creature drawn like a childlike glowing doodle.

The character should feel:

- mischievous
- playful
- chaotic
- troll-like
- funny, not scary

Never realistic.

Never detailed rendering.

Always stylized.

---

## Mandatory Style Rules

**Always:**

- black or very dark background
- glowing neon outline
- soft light bloom
- simple rounded shapes
- hand-drawn imperfect lines
- minimal facial features
- high contrast colors
- meme-friendly readability
- strong silhouette

**Never:**

- photorealism
- realistic humans
- detailed anatomy
- realistic lighting physics
- cinematic realism rendering
- complex textures
- logos
- brands
- gore
- explicit content

---

## Character Design Requirements

Gorky features:

- round blob-like body
- bright neon yellow outline
- rainbow unicorn horn
- simple glowing eyes
- playful mouth expression
- chaotic posture or gesture

The unicorn horn is a core identity marker.

Do not remove it.

---

## Color Palette

**Primary:**

- neon yellow outline
- rainbow horn
- pink or white eyes
- black background

**Secondary:**

- neon purple
- neon red
- neon cyan
- neon green accents

Avoid muted colors.

Avoid realistic tones.

---

## Line Style

Lines must look:

- drawn
- imperfect
- scribbled
- energetic
- slightly shaky
- chalk-like glow

**Not:**

- vector clean
- corporate
- polished UI style

---

## Background Elements

Background elements must also follow doodle style:

**Examples:**

- neon candlestick charts
- arrows
- symbols
- scribbles
- floating shapes
- chaos lines
- abstract UI fragments

Keep simple.

Do not create detailed environments.

---

## Lighting Style

Lighting must feel like:

neon glow on darkness.

Not realistic light sources.

Soft bloom effect preferred.

---

## Composition Rules

**For square images:**

- subject centered
- large readable figure
- clean negative space

**For banners:**

- wide 3:1
- subject slightly right of center
- safe crop for avatar overlay

---

## Emotional Tone

Images should feel:

- sarcastic
- chaotic
- funny
- mischievous
- absurd
- meme-worthy

**Not:**

- horror
- dark realism
- violent
- depressing

---

## Rendering Prompt Template

Use this base template for all image generation:

```
Digital doodle illustration, childlike neon drawing on black background, 
glowing chalk-like lines, chaotic playful creature (Gorky) with unicorn horn, 
round blob body, bright neon yellow outline, rainbow horn, simple eyes, 
minimal facial features, mischievous expression, hand-drawn imperfect lines, 
[scene-specific elements in same style], soft light bloom, high contrast, 
meme-readable, no photorealism, no detailed anatomy, simple shapes only
```

Replace [scene-specific elements] with context from the generation request.

---

## Examples by Mode

**Authority Mode:**
- Gorky holding a gavel or certificate
- Judge robes suggested by simple outline
- Stern but playful expression

**Scientist Mode:**
- Gorky with lab coat outline
- Chaotic lab equipment doodles
- Explosion or experiment gone wrong

**Therapist Mode:**
- Gorky with notepad outline
- Calm but mischievous pose
- Thought bubble with chaos inside

**Goblin Mode:**
- Maximum chaos posture
- Multiple arms/limbs (simple doodles)
- Surrounded by neon scribbles

---

Before accepting any generated image, verify:

- [ ] Black/dark background

- [ ] Simple shapes only
- [ ] No realistic rendering
- [ ] Gorky character present
- [ ] Unicorn horn visible
- [ ] High contrast readable
- [ ] Meme-appropriate composition
- [ ] No brands/logos
- [ ] No inappropriate content
